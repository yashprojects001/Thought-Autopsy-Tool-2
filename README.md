# 🌌 Overview

**Thought Autopsy Tool** is a psychology-inspired interactive web application that analyzes user thoughts in real time and visualizes emotional states through dynamic color transitions and insights.

Users write their thoughts, and the system:
- 🔍 Analyzes emotional patterns  
- 🎨 Changes UI color based on detected mental state  
- 🧠 Displays psychological insights  
- 💬 Encourages mindful reflection  

> Understand your thoughts. Visualize your emotions. Reflect with clarity.

---

# 🧠 Core Concept

The application is based on the idea of **Cognitive Reflection + Color Psychology**, where written thoughts are analyzed using keyword-driven emotional logic to simulate a lightweight AI mental analysis experience.

Each thought is processed into:
- Emotion classification 🧩  
- Color-mapped mental state 🌈  
- Insightful reflection guidance 💡  

Inspired by:
> Mental Wellness Apps • Mood Trackers • AI Journaling Tools • Reflection Systems
